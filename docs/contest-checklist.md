# Contest Checklist

## Required Submission Artifacts
- [ ] 150-word project summary (see `docs/contest-summary.md`)
- [ ] Demo video (<=3 minutes)
- [ ] Working code in GitHub repo
- [ ] README with setup and architecture
- [ ] Architecture diagram included in repo (`docs/Architecture.png`)
- [ ] 1-2 slide presentation with business value + architecture
- [ ] Link to repo included in the presentation

## Required Repo Structure
- [x] `/src` working application code
- [x] `/docs` folder exists
- [x] `AGENTS.md` added
- [x] `mcp.json` at repo root
- [ ] `/presentations/YourSolutionName.pptx` or public post link

## Scoring Coverage Plan
### 1) Enterprise applicability, reusability & business value (30)
- [ ] Define 3 enterprise scenarios and measurable KPIs
- [ ] Add before/after impact metrics in deck and README

### 2) Integration with Azure/Microsoft solutions (25)
- [ ] Document Foundry + APIM + MCP integration flow
- [ ] Add architecture callouts for each integration

### 3) Operational readiness (15)
- [ ] Deployment runbook and environment matrix
- [ ] Observability dashboard/log strategy
- [ ] CI/CD path and rollback notes

### 4) Security, governance & Responsible AI (15)
- [ ] Threats, controls, and secret management notes
- [ ] RAI safeguards and escalation process

### 5) Storytelling & amplification quality (15)
- [ ] Clear business narrative in 1-2 slides
- [ ] Crisp live demo script with outcomes

## Bonus Points
- [ ] Work IQ / Fabric IQ / Foundry IQ usage evidence
- [x] Customer validation highlights documented (`docs/operational-readiness.md`)
- [ ] Signed testimonial release form (if available)
- [ ] Copilot SDK feedback screenshot and notes
